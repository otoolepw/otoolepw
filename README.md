## Portfolio

All repositories are private. The projects below represent active and completed work across a consistent Python-dominant stack.

### Copilot Data Ownership Suite
**cpl-loader** · **cpl-viewer**

A complete end-to-end pipeline for taking ownership of Microsoft Copilot conversation history. The loader is a Windows-native ETL tool; the viewer is a full-stack web application with full-text search, topic analysis and a monthly stats dashboard. Built for personal data sovereignty — no third-party services involved.

### Homelab Utility Suite
**GMARC** · **Metrix**

**GMARC** is a forensic-grade Gmail archiving desktop application with a comprehensive analytics dashboard. Privacy-first by design — external images blocked, JavaScript disabled, HTML sanitised. Packaged as a standalone Windows executable.

**Metrix** is a household utility tracker deployed live on a homelab LXC container. Parses energy bills automatically, tracks meter readings and surfaces consumption trends via interactive dashboards. Multi-utility architecture supporting electricity and gas.

### Personal Network Toolkit
**PKTrace** · **NetStat++**

Complementary network visibility tools. PKTrace operates at the packet level; NetStat++ monitors active connections at the OS and process level. Together they provide a full-stack view of network activity on a Windows machine.

### Epistemic Toolkit
**Calibre** · **Veritas**

Two tools for evaluating information quality. Calibre assesses the psychological impact style of communicators using AI. Veritas is a structured framework for evaluating the trustworthiness of documentary-style media. Both are deterministic, scored and persist results for longitudinal analysis.

### Standalone Utilities
**WordPad++** · **SQLsnip** · **Syntrax** · **ClassDoc** · **RecTrax**

A set of practical tools filling genuine gaps. WordPad++ is a modern multi-tab rich text editor replacing the application Microsoft removed from Windows 11. SQLsnip is a system-tray SQL snippet injector for SSMS. Syntrax is a cross-device music synchronisation engine. ClassDoc is a semantic document classification and clustering system. RecTrax is a supermarket receipt analytics engine with full coverage of the Irish grocery market.

### Commercial SaaS

A B2B SaaS product in active commercial development. Full-stack FastAPI and PostgreSQL backend, deployed to cloud infrastructure.
